# 🚀 Швидка Інструкція: Створення APK для HandyHub

## ⚡ НАЙПРОСТІШИЙ СПОСІБ (через Expo Application Services)

### Крок 1: Створіть Expo акаунт (БЕЗКОШТОВНО)
```bash
npx expo register
# Або якщо вже є акаунт:
npx expo login
```

### Крок 2: Створіть APK
```bash
cd /app/frontend
npx eas build --platform android --profile production
```

### Крок 3: Зачекайте 15-20 хвилин
- EAS створить APK на своїх серверах
- Ви отримаєте посилання на завантаження
- APK буде підписаний і готовий до використання

### Крок 4: Завантажте і поділіться
- Завантажте APK з посилання
- Поділіться файлом з користувачами
- **Всі ролі (клієнт, адмін, виконавець) використовують ОДИН APK!**

---

## 🎯 ВАЖЛИВО: Один додаток = Всі ролі

### Як це працює:

```
📦 HandyHub.apk (один файл для всіх)
     ↓
┌────────────────────────────────┐
│  При реєстрації/вході:        │
│                                │
│  👤 Обрав "Клієнт"            │
│    → бачить сервіси і букінги │
│                                │
│  🔧 Обрав "Виконавець"        │
│    → бачить завдання і портфоліо│
│                                │
│  👨‍💼 Обрав "Адмін"           │
│    → бачить всю систему       │
└────────────────────────────────┘
     ↓
🌐 Всі підключені до одного Backend
     ↓
💾 Одна база даних MongoDB
```

### Синхронізація:
- **Клієнт** створює завдання → **Адмін** бачить його одразу
- **Адмін** призначає **Виконавця** → **Виконавець** отримує нотифікацію
- **Виконавець** закриває завдання → **Клієнт** бачить оновлення
- **Всі дані синхронізуються в реальному часі!**

## 📱 Встановлення APK на телефон

### Для Android:
1. Завантажте `HandyHub.apk`
2. Відкрийте файл
3. Дозвольте "Встановлення з невідомих джерел"
4. Натисніть "Встановити"
5. Готово!

### Перший запуск:
1. Відкрийте HandyHub
2. Оберіть "Зареєструватися"
3. **Оберіть роль:** Клієнт / Виконавець / Адмін
4. Заповніть дані
5. Увійдіть в систему

## ❓ FAQ

### Q: Скільки APK файлів потрібно?
**A: ОДИН!** Всі ролі в одному додатку.

### Q: Як користувачі обирають роль?
**A:** При реєстрації є вибір ролі (Клієнт/Виконавець/Адмін).

### Q: Чи бачать користувачі дані інших?
**A:** Ні! Кожна роль бачить тільки свої дані:
- Клієнт → свої букінги
- Виконавець → свої завдання
- Адмін → всі дані

### Q: Як оновити додаток?
**A:** Створіть новий APK і поділіться ним, або використайте OTA updates:
```bash
npx eas update --branch production
```

### Q: Чи потрібен окремий сервер?
**A:** Ні! Ваш backend вже працює на:
```
https://handyhub-preview-1.preview.emergentagent.com
```
